## <a name="introduction">✨ Introduction</a>

This project is an **AI-powered Resume Analyzer** designed to evaluate resumes, generate ATS scores, and provide personalized job feedback.  

Built with **React**, **React Router v7**, **TypeScript**, and **Tailwind CSS** for a clean UI, it also leverages **Puter.js** for serverless authentication, storage, and AI services.

The goal: **help candidates improve resumes and match jobs smarter.**

---

## <a name="tech-stack">⚙️ Tech Stack</a>

- **[React](https://react.dev/)** — UI library for building reusable, component-driven apps.
- **[React Router v7](https://reactrouter.com/)** — modern routing with loaders, nested routes, SSR.
- **[Puter.js](https://puter.com/)** — serverless auth, storage, and AI directly in the browser.
- **[Tailwind CSS](https://tailwindcss.com/)** — utility-first CSS framework for modern, responsive design.
- **[TypeScript](https://www.typescriptlang.org/)** — type safety + better tooling for scalable apps.
- **[Vite](https://vite.dev/)** — fast dev server & bundler.
- **[Zustand](https://github.com/pmndrs/zustand)** — lightweight global state management.

---

## <a name="features">🔋 Features</a>

👉 **Authentication** — simple & secure, powered by Puter.js.  
👉 **Resume Upload & Storage** — safe, reliable storage of resumes.  
👉 **AI Resume Matching** — get ATS scores + detailed feedback per listing.  
👉 **Modern UI/UX** — responsive design with Tailwind + shadcn/ui.  
👉 **Reusable Components** — modular, scalable architecture.  
👉 **Cross-Device Support** — works smoothly across desktop & mobile.
